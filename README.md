This repository contains an end-to-end E-Commerce Sales Analytics project using Python, SQL, and Power BI. The goal is to analyze sales performance, customer behavior, category trends, and overall business insights using real-world analytics techniques.

This project demonstrates the full workflow of a Data Analyst / Business Analyst / Power BI Developer including:

Data Cleaning (Python)

SQL Querying

Exploratory Data Analysis

Building a Power BI Dashboard

Generating Insights & Recommendations
